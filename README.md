# MedDirectory
Aplicación Web de Directorio médico

## Instalación:
### Clonar repositorio:
```
git clone https://github.com/INF331-project/MedDirectory.git
```

Luego, seguir los siguientes pasos
```
cd MedDirectory
```

Instalación de librerias y dependencias necesarias:
```
npm install
```

### Correr backend:
```
npm run dev_back
```

### Correr frontend:
```
npm run dev
```
